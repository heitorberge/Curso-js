# 📚 Wiki: Repositório Curso-js

Este repositório no GitHub, criado por Heitor Berg, é dedicado a materiais e resoluções de exercícios do **Curso de JavaScript do Curso em Vídeo**, ministrado pelo Professor Gustavo Guanabara. 🚀

O JavaScript é uma linguagem de programação essencial para o desenvolvimento web, permitindo a criação de interatividade em páginas e aplicações. Este repositório serve como um recurso prático para acompanhar e revisar os conceitos ensinados no curso.

## 🌟 Conteúdo Esperado do Repositório:

O repositório `Curso-js` contém exemplos de código, exercícios e soluções que abordam os tópicos essenciais do JavaScript, seguindo a estrutura do curso oficial do Curso em Vídeo:

* **Fundamentos do JavaScript:**
    * Introdução à linguagem e suas capacidades.
    * Sintaxe básica, variáveis (`var`, `let`, `const`), tipos primitivos.
    * Operadores (aritméticos, lógicos, relacionais, de atribuição).
    * Entrada e saída de dados.
* **Estruturas de Controle:**
    * **Condicionais (`if`, `else`, `else if`, `switch`):** Tomada de decisões.
    * **Repetições (`for`, `while`, `do-while`):** Laços para executar código repetidamente.
* **Manipulação do DOM (Document Object Model):**
    * Entendimento da estrutura de documentos HTML/CSS.
    * Seleção de elementos HTML.
    * Manipulação de conteúdo e estilos.
    * Trabalhando com eventos (clique, passagem do mouse, etc.).
* **Funções:**
    * Criação e utilização de funções.
    * Funções anônimas e Arrow Functions (ES6+).
* **Estruturas de Dados Compostas:**
    * **Arrays (Vetores):** Armazenamento e manipulação de listas de dados.
    * **Objetos:** Representação de entidades com propriedades e métodos.
* **Conceitos Avançados (ES6+):**
    * Tratamento de erros.
    * Promises e Fetch API para requisições assíncronas.
    * Async/Await para simplificar código assíncrono.
    * Módulos JavaScript.
    * Classes para Programação Orientada a Objetos.
* **Exercícios Práticos:**
    * Série de problemas resolvidos para fixação do conteúdo, que podem envolver manipulação do DOM e lógica de programação.

## 💡 Como Usar este Repositório:

Este repositório é um excelente complemento para quem está estudando JavaScript com o Curso em Vídeo ou deseja revisar os conceitos:

* **Acompanhamento de Aulas:** Utilize os códigos como referência enquanto assiste às aulas teóricas e práticas.
* **Prática Ativa:** Tente resolver os exercícios por conta própria e depois compare suas soluções com as do repositório.
* **Referência Rápida:** Consulte os exemplos para relembrar a sintaxe e a aplicação de conceitos específicos de JavaScript e DOM.
